Autonomous Rover
Prototype

Robotics · SolidWorks · Arduino · 3D Printing

Overview

A compact robotic rover designed for rugged terrain navigation and autonomous exploration. Built to tackle challenging off-road environments, the rover incorporates advanced suspension engineering and intelligent sensing systems to enable safe and reliable autonomous movement.

The chassis features a custom rocker-bogie suspension system inspired by NASA Mars rover designs, adapted for lightweight 3D-printed construction. Modular sensor mounts allow flexible integration of LIDAR, ultrasonic, and camera modules. An Arduino-based embedded controller runs real-time obstacle avoidance algorithms for fully autonomous navigation.

Key Highlights
  • Custom rocker-bogie suspension for rugged terrain stability
  • Modular mounts for LIDAR and ultrasonic obstacle sensing
  • Lightweight 3D-printed chassis optimized for mass reduction
  • Arduino-based embedded controller with real-time algorithms
  • Fully autonomous obstacle detection and avoidance
  • Scalable payload and sensor integration architecture
